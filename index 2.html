    My Projects

    Discover my projects, where creativity meets innovation

    CARING PLATES

    Caring Plates is a web-based project designed to reduce food waste and support underserved communities by connecting surplus food providers with those in need. Built using HTML, CSS, JavaScript for frontend, and Node.js with MySQL for backend, it enables donors like restaurants and households to list excess food, while NGOs and individuals can request it in real-time. With features like geolocation-based matching and secure food handling guidelines, it ensures efficient redistribution. The system integrates API calls to store donation details, making the process seamless and impactful in addressing food insecurity.

    VIP DHARISHANAM

    VIP Dharshan Booking is a web-based platform designed to help devotees seamlessly reserve priority access to temple visits. Built using Python (Flask) for backend and HTML, CSS, JavaScript for frontend, the system allows users to select temple locations, preferred dates, and VIP packages. It stores booking details in SQLite or MySQL, ensuring smooth management of reservations. Secure payment integration ensures hassle-free transactions, while automated confirmation improves user experience. The platform streamlines booking processes, reducing wait times and enhancing accessibility for spiritual seekers.
